Ayuda de formulario

9: Marque o demarque para determinar si tiene o no alcohol
10: Pulse sobre el calendario para seleccionar un fecha
11: Pulse en la imagen para añadir un nuevo campo tipo, o pulse sobre la flecha para ver un listado de los actuales.
12: Pulse para guardar el vino en el storage
13: Pulse para borrar
